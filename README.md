## Getting started

### Hugo

Install [Hugo](https://gohugo.io/):

```sh
brew install hugo
```

Clone the repository:

```sh
git clone https://github.com/riceset/riceset.com && cd riceset.com
```

finally, start the development server with:

```sh
hugo server -t hello-friend
```

## Mentioned

- [Hugo](https://gohugo.io/)
